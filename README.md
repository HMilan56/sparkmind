# SparkMind

This project is a quiz application developed as part of the project laboratory university course.

The primary goal of this project is to develop a full-stack application while exploring a wide range of modern technologies. The application is inspired by Kahoot!, featuring a competitive, fast-paced environment where multiple players can join, answer questions simultaneously, and see their rankings update in real-time.

## Tech stack
- Frontend: Typescript, React, MUI, Vite
- Backend: C#, ASP.NET, SignalR, PostgreSQL

## Key features

- Live lobbies: Players can join using a PIN code.
- Quiz Creator: A dedicated interface for users to build their own custom quizzes.
- Timed Questions: Fast-paced gameplay with configurable countdowns.

## Roadmap
